📅 Google Kalendere

Tilføj ICS-links fra Google Kalender eller Apple Kalender (iCloud).

📖 Sådan finder du dit ICS-link
🟦 Google Kalender
  1. Åbn calendar.google.com
  2. Klik på de tre prikker ud for din kalender → Indstillinger og deling
  3. Scroll ned til Integrer kalender
  4. Kopiér Offentlig adresse i iCal-format
🍎 Apple Kalender (iCloud)
  1. Kalender-appen → tryk ℹ️ ud for din kalender
  2. Slå Offentlig kalender til → kopiér URL
  3. Erstat webcal:// med https://

🔑 API Nøgle

Beskytter dashboardet mod uautoriseret adgang på dit netværk.

Forbedrer dato-parsing fra Aula-beskeder. Uden nøgle bruges en simpel dansk parser. Hent nøgle på console.anthropic.com.

🔐 Google Kalender API (OAuth2)

Tilslut din Google-konto for automatisk adgang til alle dine kalendere — inkl. private. Client ID og Secret er skjulte.

📖 Opsætningsguide — klik for at folde ud/ind
Trin 1 — Opret Google Cloud projekt
Gå til console.cloud.google.com → klik Select project → New Project → giv det et navn → Create.
Trin 2 — Aktivér Google Calendar API
Gå til APIs & Services → Library → Google Calendar API → vælg dit projekt → klik Enable.
Trin 3 — OAuth Consent Screen
Gå til Google Auth Platform → Branding:
1. Vælg External → Create
2. Udfyld App name (f.eks. "Hjemmedashboard"), Support email og Developer contact email
3. Klik Save
4. Tilføj Scopes — gå til Google Auth Platform → Data Access:
  • Klik Add or Remove Scopes
  • Søg efter calendar.events
  • Vælg https://www.googleapis.com/auth/calendar.readonly
  • Klik Save
Trin 4 — Tilføj test-bruger
Gå til Google Auth Platform → Audience (Google har omdøbt "OAuth consent screen" til dette):
1. Scroll ned til Test users
2. Klik + Add users
3. Tilføj din Gmail-adresse (f.eks. din@gmail.com) → Save
⚠️ Uden dette får du "Access blocked" fejl ved login.
Trin 5 — Opret OAuth Client ID
Gå til Google Auth Platform → Clients (tidligere under "APIs & Services → Credentials"):
1. Klik + Create Client
2. Application type: Web application
3. Under Authorized redirect URIs → klik + Add URI → indtast:
http://localhost:8000/auth/google/callback
4. Klik Create — du får nu Client ID og Client Secret.
Trin 6 — Indsæt og tilslut
Kopiér Client ID og Client Secret ind i felterne nedenfor → klik Gem indstillinger → klik Tilslut Google-konto → log ind og godkend.
Fra Google Cloud Console → Credentials → OAuth 2.0 Client ID
Nye aftaler oprettes i denne Google-kalender

📅 Kalenderabonnement (ICS)

Abonnér på lokale begivenheder i din kalender-app.

📖 Sådan tilføjer du abonnementet
🍎 Apple Kalender
  1. Indstillinger → Kalender → Konti → Tilføj konto → Andet
  2. Vælg Tilføj abonnementskalender → indsæt URL
🤖 Android (ICSx⁵)
  1. Installér ICSx⁵ fra F-Droid eller Play Store
  2. Tilføj abonnement → indsæt URL → vælg synkroniseringsinterval
🟦 Google Kalender

⚠️ Kræver at din server er tilgængelig fra internettet — virker ikke på lokalt netværk.
